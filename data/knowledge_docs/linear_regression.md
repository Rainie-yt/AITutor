# 线性回归

## 1. 基本定义

线性回归是一种典型的监督学习算法，主要用于回归任务。它通过学习输入特征和输出结果之间的线性关系，对连续数值进行预测。

在《人工智能导论》中，线性回归可以作为理解“监督学习”“损失函数”和“模型训练”的基础算法。

## 2. 核心思想

线性回归的核心思想是：找到一条直线或一个线性函数，使模型预测值尽可能接近真实值。

简单来说，线性回归希望通过已有数据学习出一个规律，然后用这个规律预测新的结果。

例如，根据房屋面积预测房价，根据学习时间预测考试成绩，根据广告投入预测销售额，都可以看作回归问题。

## 3. 模型形式

简单线性回归可以表示为：

y = wx + b

其中：

- x 表示输入特征；
- y 表示预测结果；
- w 表示权重参数；
- b 表示偏置项。

如果有多个输入特征，可以写成多元线性回归：

y = w_1x_1 + w_2x_2 + ... + w_nx_n + b

其中，x_1, x_2, ..., x_n 表示不同的输入特征，w_1, w_2, ..., w_n 表示对应的权重参数。

## 4. 损失函数

线性回归常用均方误差作为损失函数，用来衡量预测值和真实值之间的差距。

损失函数可以表示为：

J = 1/m Σ(y_i - ŷ_i)²

其中：

- m 表示样本数量；
- y_i 表示第 i 个样本的真实值；
- ŷ_i 表示第 i 个样本的预测值；
- J 表示整体预测误差。

损失函数越小，说明模型预测结果越接近真实结果。

## 5. 参数求解方法

线性回归的目标是找到合适的参数 w 和 b，使损失函数尽可能小。

常见的参数求解方法包括：

1. 最小二乘法：通过数学推导直接求解最优参数；
2. 梯度下降：通过不断更新参数，使损失函数逐渐减小。

在机器学习中，梯度下降更常用于理解模型训练过程。

## 6. 典型应用

线性回归常用于以下场景：

- 房价预测；
- 成绩预测；
- 销量预测；
- 股票趋势分析；
- 广告投入与收益分析；
- 其他连续数值预测任务。

## 7. 常见误区

线性回归主要用于预测连续数值，不适合直接处理分类问题。

线性回归中的“线性”主要指模型参数之间是线性关系，并不一定要求所有数据在图像上完全呈现直线分布。

另外，线性回归的效果会受到异常值影响。如果数据中存在极端异常点，模型参数可能会被明显干扰。

## 8. 优点与局限

线性回归的优点是原理简单、容易理解、计算效率较高，并且模型结果具有较好的可解释性。

它的局限是表达能力有限，难以刻画复杂的非线性关系。如果数据之间的关系非常复杂，单纯使用线性回归可能效果不理想。
