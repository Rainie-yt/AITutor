# K-means 聚类算法

## 1. 基本定义

K-means 是一种典型的无监督学习算法，主要用于聚类任务。它的目标是将没有标签的数据划分为 K 个簇，使同一簇内的数据尽可能相似，不同簇之间的数据尽可能不同。

在《人工智能导论》中，K-means 可以作为理解“无监督学习”和“聚类思想”的代表性算法。

## 2. 核心思想

K-means 的核心思想是：先随机选择 K 个聚类中心，然后根据样本到聚类中心的距离进行分组，再根据每个簇内样本的均值更新聚类中心。

这个过程会不断重复，直到聚类中心基本不再变化，或者达到设定的最大迭代次数。

简单来说，K-means 每轮主要做两件事：

1. 按距离把样本分到最近的聚类中心；
2. 根据每一组样本重新计算新的聚类中心。

## 3. 算法步骤

K-means 的基本步骤如下：

1. 确定聚类数量 K；
2. 随机选择 K 个样本作为初始聚类中心；
3. 计算每个样本到各个聚类中心的距离；
4. 将每个样本分配到距离最近的聚类中心所在的簇；
5. 对每个簇中的样本求均值，并将该均值作为新的聚类中心；
6. 重复步骤 3 到步骤 5，直到聚类中心不再明显变化。

## 4. 核心公式

K-means 的目标是最小化样本到所属聚类中心的距离平方和，其目标函数可以表示为：

J = Σ ||x_i - μ_k||²

其中：

- x_i 表示第 i 个样本点；
- μ_k 表示第 k 个聚类中心；
- ||x_i - μ_k||² 表示样本点到聚类中心的距离平方；
- J 表示整体聚类误差。

该公式的含义是：让每个样本尽可能靠近它所属簇的中心点。

## 5. 典型应用

K-means 常用于以下场景：

- 客户分群：根据用户消费行为将用户划分为不同群体；
- 图像分割：根据像素特征将图像划分为不同区域；
- 文本聚类：将相似主题的文本归为一类；
- 异常检测：发现与大多数样本差异较大的数据点；
- 数据预处理：对无标签数据进行初步分组。

## 6. 常见误区

K-means 容易和 KNN 混淆，但二者并不相同。

K-means 是无监督学习算法，主要用于聚类任务，不需要样本标签。

KNN 是监督学习算法，常用于分类或回归任务，需要已知标签的训练数据。

另外，K-means 中的 K 表示聚类数量，而 KNN 中的 K 表示参与投票或预测的邻居数量。

## 7. 优点与局限

K-means 的优点是原理简单、实现方便、计算速度较快，适合处理较大规模的数据。

K-means 的局限是需要提前指定 K 值，并且对初始聚类中心和异常值比较敏感。如果初始中心选择不同，最终聚类结果可能会有所不同。

同时，K-means 更适合处理近似球形分布的数据。如果数据分布形状复杂，聚类效果可能不理想。
